# 🚀 NexBean Chatbot

An intelligent chatbot application powered by modern web technologies and AI integration.
Built with a scalable architecture using Next.js frontend and Node.js backend.

---

## 🌐 Live Demo

🔗 https://nexbean-chatbot-yuqc-3whsp7u1g-nexbeans-projects.vercel.app/auth/login

---

## ✨ Features

* 🤖 AI-powered chatbot (Gemini API)
* 🔐 Secure authentication (JWT + bcrypt)
* ⚡ Fast and responsive UI
* 🎨 Modern design with Tailwind CSS
* 🔔 Real-time notifications using React Toastify
* 🔄 State management using Context API
* 🌐 API integration with Axios
* 📱 Fully responsive design

---

## 🧱 Tech Stack

### 🖥 Frontend

* Next.js
* React
* Tailwind CSS
* Axios
* React Toastify
* Context API

### ⚙️ Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* bcrypt (Password Encryption)
* Gemini API (AI Chatbot)

## ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/vishnuj756/nexbean-chatbot.git
cd nexbean-chatbot
```

---

### 2️⃣ Setup Frontend

```bash
cd FE
npm install
npm run dev
```

---

### 3️⃣ Setup Backend

```bash
cd BE
npm install
npm run start
```

---

## 🔐 Environment Variables

Create a `.env` file in both frontend & backend.

### Backend (.env)

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_api_key
```

### Frontend (.env)

```env
NEXT_PUBLIC_API_URL=https://your-backend-url
```

## 🚀 Deployment

* Frontend → Deployed on Vercel
* Backend → Deployed on Render

---

## 🧠 How It Works

1. User logs in securely using JWT authentication
2. Passwords are encrypted using bcrypt
3. Frontend sends requests using Axios
4. Backend processes requests and interacts with Gemini API
5. Chatbot responses are returned and displayed in real-time
